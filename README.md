# ALT Linux Package Comparison CLI

## Installation 

1. Clone repository
    ```
    git clone https://github.com/Lunovoy/task-bazalt.git
    cd task-bazalt
    ```
2. Build

    ```
    go build ./cmd/comparison.go
    ```
3. Run
    >Usage:
    >comparison "branch1" "branch2"

    Example:
    ```
    comparison p10 p9
    ```
